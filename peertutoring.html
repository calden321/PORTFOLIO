Peer Tutoring Mobile App

An Android application developed to connect students for peer-to-peer tutoring sessions at DUT, featuring tutor availability scheduling, resource sharing, and a dual rating system.

Tutor Experience Flow

Welcome Screen

Initial landing page with motivational quote about knowledge sharing

Login Screen

Secure login for returning tutors with DUT email authentication

Tutor Registration

Comprehensive sign-up process with role selection and academic details

Tutor Profile Setup

Section for tutors to describe their experience and set hourly rates

Availability Management

Interactive interface for tutors to set their available time slots

Tutor Dashboard

Main hub with access to all tutor-specific features and functions

Resource Upload

Interface for tutors to share educational materials with students

Tutee Experience Flow

Tutee Dashboard

Personalized dashboard showing tutor availability and quick access to features

Feedback System

Rate tutors and provide feedback after sessions

Resource Library

Access study materials organized by faculty and academic year

TutorBot Assistant

AI chatbot to help with common questions and navigation

Find a Tutor

Browse and search for tutors by faculty, course, and availability

Tutor Details

View tutor details, ratings, and available time slots

Session Confirmation

Confirmation screen after successfully booking a tutoring session

Key Features

Availability Scheduling

Tutors can set available time slots, tutees can view and book sessions.

Resource Sharing

Tutors upload study materials, tutees can download by faculty/course.

Dual Rating System

Both tutors and tutees can rate each other after sessions.

TutorBot Assistant

AI-powered assistant to help with navigation and common questions.

Tutor Discovery

Advanced search and filtering to find the perfect tutor.

Session Booking

Easy booking system with instant confirmation.
